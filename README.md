# SkillBarter — Full Stack Web Application

## Overview

SkillBarter is a full-stack web application that allows users to exchange skills with others. Users can create profiles, list their skills, connect with others, and communicate through a messaging system.

This project demonstrates modern full-stack development using React for the frontend and Supabase as a Backend-as-a-Service (BaaS).

---

## Tech Stack

### Frontend

* React (Vite)
* JavaScript (ES6+)
* CSS

### Backend (BaaS)

* Supabase (PostgreSQL, Authentication, Realtime)

### Deployment

* Frontend: GitHub Pages
* Backend Services: Supabase Cloud

## Architecture

```text
Frontend (React)
      ↓
Supabase (Auth + Database + API)
      ↓
PostgreSQL Database
```

Supabase handles:

* Authentication
* Database queries
* Realtime updates
